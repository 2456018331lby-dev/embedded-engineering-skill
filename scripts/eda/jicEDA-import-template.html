📋 导入步骤

1 打开嘉立创EDA

访问 lceda.cn/editor 或打开嘉立创EDA客户端

2 新建原理图

点击 文件 → 新建 → 原理图

3 打开控制台

按 F12 打开浏览器开发者工具，切换到 Console（控制台）标签

4 复制并执行代码

点击下方按钮复制导入代码，然后粘贴到控制台执行：

⚠️ 注意：
  • 确保在嘉立创EDA的 原理图编辑器 中执行，不是PCB编辑器
  • 执行后可能需要等待几秒钟加载
  • 如果报错，请刷新页面重试

🔧 备用方案

如果控制台方式不工作：

  1. 下载 JSON 文件
  2. 在嘉立创EDA中点击 文件 → 导入 → EasyEDA
  3. 选择下载的 JSON 文件
