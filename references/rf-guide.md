## 1. 微带线设计

### 1.1 基本概念

微带线是PCB上最常用的传输线结构：顶层导体条 + 介质基板 + 底层地平面。

特征阻抗由线宽(W)、基板厚度(H)、介电常数(εr)决定。

### 1.2 阻抗计算公式

**当 W/H ≤ 1 时：**
```
Z0 = (60 / √εeff) × ln(8H/W + W/(4H))
εeff = (εr+1)/2 + (εr-1)/2 × (1 + 12H/W)^(-0.5)
```

**当 W/H > 1 时：**
```
Z0 = (120π / √εeff) / (W/H + 1.393 + 0.667×ln(W/H + 1.444))
εeff = (εr+1)/2 + (εr-1)/2 × (1 + 12H/W)^(-0.5)
```

### 1.3 常用基板50Ω线宽速查

| 基板材料 | εr | 厚度(mm) | 50Ω线宽(mm) | 适用频段 |
|---------|-----|---------|------------|---------|
| FR4 | 4.4 | 1.6 | 2.9 | DC~3GHz |
| FR4 | 4.4 | 0.8 | 1.4 | DC~5GHz |
| Rogers 4003C | 3.55 | 0.8 | 1.8 | DC~20GHz |
| Rogers 4350B | 3.66 | 0.76 | 1.7 | DC~20GHz |
| Rogers 5880 | 2.2 | 0.787 | 2.4 | DC~40GHz |
| Taconic TLY | 2.2 | 0.787 | 2.4 | DC~40GHz |

### 1.4 微带线设计注意事项

**线宽公差**
- PCB制造公差通常 ±0.05mm，高频设计需要向厂家确认
- 线宽越窄，公差影响越大，建议 W > 0.2mm

**损耗来源**
- 导体损耗：与频率平方根成正比，高频主要损耗
- 介质损耗：与 tanδ 成正比，FR4 tanδ≈0.02，Rogers tanδ≈0.001
- 辐射损耗：频率越高越显著

**过孔影响**
- 每个过孔引入约 0.5~1nH 电感
- 高频设计尽量避免在射频路径上打过孔
- 必须过孔时，在过孔周围加地过孔阵列

### 1.5 共面波导（CPW）

当需要更好的屏蔽或更低的辐射时，使用共面波导：
- 顶层：信号线 + 两侧地铜
- 间距 G 和线宽 W 共同决定阻抗
- 优点：无需参考地平面，适合单层板

## 2. 天线设计

### 2.1 天线基本参数

| 参数 | 说明 |
|------|------|
| 增益(dBi) | 相对于各向同性天线的方向性增益 |
| 方向图 | 天线辐射的空间分布 |
| 带宽 | S11 < -10dB（VSWR < 2）的频率范围 |
| 极化 | 线极化（水平/垂直）或圆极化 |
| 效率 | 辐射功率/输入功率，含匹配损耗和导体损耗 |
| VSWR | 电压驻波比，理想=1，实用<2 |

### 2.2 常见天线类型与设计

#### PCB倒F天线（PIFA / IFA）
适用：2.4GHz WiFi/BT、433MHz、868MHz IoT模块内置天线

设计要点：
- 2.4GHz PIFA：辐射臂长约 31mm（λ/4），馈电点距短路点约 3~5mm
- 净空区（Clearance）：天线下方及周围不能有铜，至少 3~5mm
- 地平面尺寸影响谐振频率，地越大频率越低
- 调谐：改变辐射臂长度（±1mm 约影响 ±50MHz @ 2.4GHz）

```
典型2.4GHz IFA结构（俯视）：
                    辐射臂（~31mm）
    短路点          ┌──────────────────────┐
    │               │                      │
────┘               │                      │
地平面              馈电点
```

#### 偶极子天线
适用：通用宽带，外置天线

- 全波长偶极子增益：2.15dBi
- 半波偶极子长度：λ/2 = 150/f(MHz) mm
- 2.4GHz 半波偶极子：约 62mm（每臂 31mm）
- 馈电阻抗：约 73Ω，需匹配到 50Ω

#### 贴片天线（Patch Antenna）
适用：GPS(1575MHz)、2.4GHz 固定安装

- 谐振频率：f = c / (2L√εr)，L 为贴片长度
- GPS 标准贴片：25×25mm（陶瓷基板）
- 需要有源放大（LNA）配合使用
- 注意：贴片天线对安装面敏感，金属面会失谐

#### Yagi-Uda 天线
适用：定向通信，UHF/微波频段

- 增益：每增加一个引向器约 +1~2dBi
- 3单元Yagi：约 7dBi
- 5单元Yagi：约 10dBi
- 反射器比振子长约 5%，引向器比振子短约 5%

#### 螺旋天线
适用：卫星通信、圆极化需求

- 轴向模式（端射）：圆极化，高增益
- 法向模式（边射）：线极化，低剖面
- 轴比（AR）< 3dB 为圆极化

### 2.3 天线净空区规范

**关键原则**：天线净空区内不能有任何铜（包括地铜、走线、过孔）

| 频段 | 最小净空区 |
|------|-----------|
| 2.4GHz | 3mm × 15mm |
| 868/915MHz | 5mm × 20mm |
| 433MHz | 10mm × 30mm |

### 2.4 天线调试流程

1. VNA 测量 S11，观察谐振点
2. 谐振频率偏高 → 加长辐射臂
3. 谐振频率偏低 → 缩短辐射臂
4. 带宽不够 → 加宽辐射臂或使用匹配网络展宽
5. S11 最低点不够低（> -10dB）→ 调整馈电点位置
6. 最终目标：目标频率处 S11 < -10dB，带宽覆盖整个工作频段

## 3. Smith圆图与阻抗匹配

### 3.1 Smith圆图基础

Smith圆图是射频工程最核心的工具，将复数阻抗映射到反射系数平面。

**坐标系**
- 圆心：Z0（归一化阻抗 = 1，即50Ω系统中的50Ω）
- 最左端：短路（Γ = -1）
- 最右端：开路（Γ = +1）
- 上半圆：感性（+jX）
- 下半圆：容性（-jX）

**归一化阻抗**
```
z = Z / Z0 = r + jx
其中 Z0 = 50Ω（标准系统阻抗）
```

**反射系数与阻抗的关系**
```
Γ = (Z - Z0) / (Z + Z0) = (z - 1) / (z + 1)
|Γ| = 0 → 完美匹配
|Γ| = 1 → 全反射（短路或开路）
```

### 3.2 Smith圆图上的元件效果

| 操作 | Smith圆图上的轨迹 |
|------|-------------------|
| 串联电感 | 沿等电阻圆顺时针移动（向上） |
| 串联电容 | 沿等电阻圆逆时针移动（向下） |
| 并联电感 | 沿等电导圆逆时针移动（导纳圆图） |
| 并联电容 | 沿等电导圆顺时针移动（导纳圆图） |
| 传输线 | 沿等VSWR圆顺时针旋转，长度λ/2转一圈 |

### 3.3 VSWR与回波损耗换算

| VSWR | |Γ| | 回波损耗(dB) | 反射功率 |
|------|-----|-------------|---------|
| 1.0 | 0 | ∞ | 0% |
| 1.2 | 0.091 | 20.8 | 0.8% |
| 1.5 | 0.200 | 14.0 | 4.0% |
| 2.0 | 0.333 | 9.5 | 11.1% |
| 3.0 | 0.500 | 6.0 | 25.0% |
| 5.0 | 0.667 | 3.5 | 44.4% |

**实用判据**：VSWR < 2（S11 < -10dB）为可接受匹配

### 3.4 Smith圆图实战步骤

1. 用VNA测量负载阻抗 ZL，归一化得 zL = ZL/50
2. 在Smith圆图上标出 zL
3. 确定目标：移动到圆心（z = 1）
4. 选择匹配拓扑（串联/并联，电感/电容）
5. 计算元件值：
   - 电感：L = X × Z0 / (2πf)
   - 电容：C = 1 / (2πf × X × Z0)
6. 验证：在目标频段内检查匹配带宽

### 3.5 导纳圆图（Y-Smith）

并联元件在导纳圆图上更直观：
```
y = Y / Y0 = g + jb
Y0 = 1/Z0 = 0.02S（50Ω系统）
```

阻抗圆图和导纳圆图关于圆心旋转180°的关系：
- 阻抗圆图上的点 z，对应导纳圆图上关于圆心对称的点 y = 1/z

## 4. S参数基础

### 4.1 S参数定义

S参数（散射参数）描述射频网络端口间的功率传输关系，用反射波与入射波之比表示。

**二端口网络：**
```
b1 = S11·a1 + S12·a2
b2 = S21·a1 + S22·a2

其中 a = 入射波，b = 反射波
```

| 参数 | 含义 | 实用解释 |
|------|------|---------|
| S11 | 输入反射系数 | 端口1的回波损耗，越小越好 |
| S21 | 正向传输系数 | 插入损耗（无源）或增益（有源） |
| S12 | 反向传输系数 | 隔离度（放大器）或反向插损 |
| S22 | 输出反射系数 | 端口2的回波损耗 |

### 4.2 S参数与常用指标换算

**回波损耗（Return Loss）**
```
RL(dB) = -20×log10(|S11|)
```

**插入损耗（Insertion Loss）**
```
IL(dB) = -20×log10(|S21|)   （无源器件，S21 < 1）
```

**增益（Gain）**
```
G(dB) = 20×log10(|S21|)     （有源器件，S21 > 1）
```

**VSWR**
```
VSWR = (1 + |S11|) / (1 - |S11|)
```

### 4.3 S参数文件格式（Touchstone .s2p）

```
! 注释行
# GHz S MA R 50
! freq     S11_mag  S11_ang  S21_mag  S21_ang  S12_mag  S12_ang  S22_mag  S22_ang
2.400     0.150    -45.0    0.950    -12.0    0.020    160.0    0.120    -30.0
2.450     0.130    -50.0    0.940    -15.0    0.018    155.0    0.110    -35.0
```

格式说明：
- `# GHz S MA R 50`：频率单位GHz，S参数，幅度-角度格式，参考阻抗50Ω
- MA = 幅度+角度，RI = 实部+虚部，DB = dB+角度

### 4.4 S参数测量要点

**VNA校准**
- 必须校准：SOLT（Short-Open-Load-Thru）或 TRL
- 校准面要尽量靠近DUT（被测器件）
- 校准后验证：测量已知标准件

**测量注意事项**
- 连接器扭矩：SMA 用 8 in-lb 扭矩扳手
- 电缆弯曲：测量过程中不要移动电缆
- 端口延伸：如果DUT不在校准面，需要做端口延伸去嵌入

### 4.5 混合模S参数（差分信号）

差分对使用4端口S参数，转换为混合模：
- Sdd：差分到差分（关注的主要参数）
- Scc：共模到共模
- Sdc/Scd：模式转换（越小越好）

USB 3.0、PCIe、HDMI等高速差分信号需要关注Sdd21（差分插损）和Sdd11（差分回损）

## 5. 匹配网络设计

### 5.1 为什么需要匹配

- 最大功率传输：源阻抗 = 负载阻抗共轭时功率传输最大
- 减少反射：不匹配导致驻波，增加损耗和信号畸变
- 保护器件：大反射可能损坏功放等有源器件

### 5.2 L型匹配网络

最简单的匹配拓扑，两个元件（一个串联一个并联）。

**拓扑选择规则：**
- RL > Z0 → 并联元件靠近负载
- RL < Z0 → 并联元件靠近源

**计算步骤（RL > Z0 的情况）：**
```
Q = √(RL/Z0 - 1)
并联电抗：Xp = RL / Q
串联电抗：Xs = Q × Z0
```

**示例：50Ω匹配到150Ω @ 2.4GHz**
```
Q = √(150/50 - 1) = √2 ≈ 1.414
Xp = 150 / 1.414 = 106.1Ω → C = 1/(2π×2.4e9×106.1) = 0.625pF
Xs = 1.414 × 50 = 70.7Ω → L = 70.7/(2π×2.4e9) = 4.69nH
```

**局限性：** 窄带，Q值不可控

### 5.3 π型匹配网络

三个元件，提供额外自由度控制Q值（带宽）。

```
结构：
源 ──┬── L/C ──┬── 负载
     C1        C2
     │         │
    GND       GND
```

**优点：**
- 可独立控制匹配和带宽
- 适合功放输出匹配
- 天然低通滤波，抑制谐波

**设计步骤：**
1. 选择虚拟电阻 Rv < min(RS, RL)
2. 将π网络拆成两个背靠背的L网络
3. 分别计算每个L网络的元件值

### 5.4 T型匹配网络

```
结构：
源 ── L/C ──┬── L/C ── 负载
            C/L
            │
           GND
```

**特点：**
- 高通特性，不抑制谐波
- 适合需要DC隔离的场景
- Q值通常比π型高

### 5.5 单支节匹配（Stub Matching）

用传输线段代替集总元件，适合微波频段。

**短截线（Stub）类型：**
- 开路stub：等效并联电容（短于λ/4）或并联电感（长于λ/4）
- 短路stub：等效并联电感（短于λ/4）或并联电容（长于λ/4）

**单stub匹配步骤：**
1. 在Smith圆图上找到负载点
2. 沿传输线旋转到 g=1 圆（等电导圆）
3. 记录旋转距离 d（stub到负载的距离）
4. 计算stub长度使其电纳抵消负载电纳

### 5.6 四分之一波长变换器

```
Z_transformer = √(Z1 × Z2)
长度 = λ/4（在工作频率处）
```

**示例：50Ω到100Ω**
```
Z_t = √(50×100) = 70.7Ω
```

**带宽：** 约 ±10~15%（单节），多节级联可展宽

### 5.7 匹配网络元件选型

| 频段 | 电感推荐 | 电容推荐 |
|------|---------|---------|
| < 1GHz | Murata LQW系列，Coilcraft 0402HP | Murata GRM系列 C0G |
| 1~6GHz | Coilcraft 0201DS，Johanson L-07C | AVX Accu-P，Murata GJM |
| > 6GHz | 印制电感（微带线stub） | 印制电容或超小封装 |

**关键：** 高频电感必须选择高自谐振频率（SRF）的型号，SRF至少为工作频率的2倍

## 6. PCB射频布局规范

### 6.1 层叠设计

**4层板推荐叠层（射频项目）：**
```
L1: 信号层（射频走线 + 关键数字信号）
L2: 完整地平面（GND）—— 射频参考地，不可分割
L3: 电源层
L4: 信号层（低速信号）
```

**关键原则：**
- 射频走线必须紧邻完整地平面
- 地平面不能有开槽穿过射频走线下方
- 电源层和地层之间加去耦电容

### 6.2 射频走线规则

**阻抗控制**
- 50Ω微带线：线宽由叠层决定（见第1章速查表）
- 走线宽度全程一致，避免突变
- 拐角用圆弧或45°斜角，禁止90°直角

**间距规则**
- 射频线间距 ≥ 3倍线宽（减少耦合）
- 射频线到数字信号 ≥ 5倍线宽
- 射频线到板边 ≥ 3倍线宽

**长度控制**
- 射频走线越短越好
- 每增加1cm（FR4 @ 2.4GHz）约增加 0.3dB 损耗

### 6.3 接地策略

**地过孔阵列**
- 射频走线两侧每隔 λ/10 放置地过孔
- 2.4GHz：约每 5mm 一个地过孔
- 过孔直径 0.3mm，焊盘 0.6mm

**接地岛**
- 射频器件下方保持完整地铜
- 器件接地焊盘用多个过孔连接到内层地

**分区隔离**
- 射频区、数字区、电源区物理分离
- 区域间用地过孔墙隔离
- 敏感射频电路加屏蔽罩焊盘

### 6.4 连接器与过渡

**SMA连接器**
- 焊盘到微带线过渡要平滑
- 连接器接地焊盘周围密布地过孔
- 推荐：边缘安装SMA，减少过渡不连续

**U.FL/IPEX连接器**
- 适合板内天线连接
- 注意：插拔寿命约30次，不适合频繁操作
- 走线从连接器到天线保持50Ω

### 6.5 去耦与电源

**射频器件电源去耦**
```
VCC ── 磁珠(600Ω@100MHz) ── 100nF ── 器件VCC
                              │
                             GND
可选：再并联 1μF + 10pF 覆盖宽频段
```

**LDO选择**
- 射频电源用低噪声LDO（噪声 < 10μVrms）
- 推荐：TPS7A47（TI）、ADP151（ADI）
- PSRR > 60dB @ 工作频率

### 6.6 EMC注意事项

- 时钟信号远离射频区域
- 高速数字信号包地处理
- 板边加接地铜皮和过孔墙
- 射频输入输出端口加ESD保护（TVS二极管，注意寄生电容 < 0.5pF）

## 7. 仿真工具指导

### 7.1 Keysight ADS（Advanced Design System）

**适用场景：** 射频电路仿真、匹配网络优化、天线馈电网络

**常用仿真类型：**
- S参数仿真（S-Param Simulation）：频域分析，最常用
- 谐波平衡（Harmonic Balance）：非线性电路，功放、混频器
- 瞬态仿真（Transient）：时域分析

**ADS匹配网络设计流程：**
1. 新建原理图，放置S参数源（端口）
2. 插入Smith Chart Matching工具
3. 输入源阻抗、负载阻抗、频率
4. 工具自动生成匹配拓扑和元件值
5. 仿真验证S11，优化带宽

**ADS Layout联动：**
- 原理图完成后可直接生成Layout
- 用Momentum做电磁场仿真验证PCB效果
- 注意：仿真时要包含过孔、焊盘等寄生效应

### 7.2 HFSS / CST（全波电磁仿真）

**适用场景：** 天线设计、EMC分析、高频互连建模

**HFSS天线仿真流程：**
1. 建立3D模型（天线+基板+地平面+馈电）
2. 设置边界条件（辐射边界，距天线 > λ/4）
3. 设置激励（波端口或集总端口）
4. 设置频率扫描范围
5. 运行仿真，查看S11、方向图、增益

**仿真收敛判据：**
- ΔS < 0.02（默认）
- 网格细化至少3次自适应通过

### 7.4 仿真与实测对比

**常见偏差来源：**
- PCB介电常数实际值与标称值差异（FR4 εr = 4.2~4.8）
- 焊接寄生效应（焊锡量、焊盘大小）
- 连接器过渡不连续
- 周围金属结构影响（外壳、螺丝）

**建议：**
- 仿真结果作为起点，实测微调
- 预留匹配网络焊盘（即使仿真显示不需要）
- 关键参数留 10~20% 余量

## 8. 常见问题排查

### 8.1 天线不谐振

| 症状 | 可能原因 | 解决方案 |
|------|---------|---------|
| S11全频段 > -5dB | 馈电点位置错误 | 调整馈电点，检查焊接 |
| 谐振频率偏高 | 辐射臂太短 | 加长辐射臂或加载电容 |
| 谐振频率偏低 | 辐射臂太长或地平面太大 | 缩短辐射臂 |
| 带宽太窄 | Q值太高 | 加宽辐射臂或用匹配网络展宽 |
| 双谐振 | 寄生模式 | 检查净空区，排除耦合 |

### 8.2 射频链路增益不足

- 检查每级增益和损耗，画增益预算表
- 确认LNA供电正常（电流是否符合数据手册）
- 检查匹配网络损耗（高Q电感损耗大）
- 确认PCB走线损耗（FR4在高频损耗显著）

### 8.3 杂散和谐波超标

- 功放输出加低通滤波器
- 检查电源纹波（开关电源噪声混入射频）
- 本振泄漏：提高混频器LO-RF隔离度
- 数字时钟谐波：远离射频区域，加屏蔽

### 8.4 接收灵敏度差

- 检查NF（噪声系数）：第一级LNA的NF决定系统NF
- 确认LNA增益足够（> 15dB）
- 检查镜像频率抑制
- 排除板级自激（检查稳定性，K > 1）
